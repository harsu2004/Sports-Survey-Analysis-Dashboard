1. Project Name:- Sports Survey Analysis Dashboard
2. Project Overview:- 

The Sports Survey Analysis Dashboard is an interactive data visualization project that provides insights into sports participation, preferences, and distribution based on survey data.

3. Objectives:- 
  - Analyze survey data related to sports participation
  - Identify most popular sports
  - Understand gender distribution across different sports
  - Examine regional (Jila-wise) participation trends
  - Visualize time preferences for sports activities

4. Technologies Used:- 
  - Power BI – for dashboard development
  - Microsoft Excel / CSV – for data cleaning and preprocessing
  - Data Visualization Techniques – charts, maps, and KPIs
  - Basic Data Analysis – aggregation, filtering, segmentation

5. Key Dashboard Insights:- 
  - Sports-wise Participation Count using pie chart
  - Time Distribution Analysis (Morning vs Evening)
  - Jila-wise Survey Count by Sports (stacked bar chart)
  - Gender-wise Participation (Boys vs Girls comparison)
  - Geographical Visualization using map for regional insights
  - Interactive Filters (State/Region Selection)
